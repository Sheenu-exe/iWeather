#iWeather - Weather Forecast Web App
![icons8-weather-120](https://github.com/Sheenu-exe/iWeather/assets/130788900/8c6212ca-cec8-40b8-8085-e72e10d3d7c6)


iWeather is a simple and user-friendly web application that provides real-time weather information for locations around the world. With a clean and intuitive interface, it allows users to quickly access up-to-date weather data including temperature, humidity, wind speed, and more.


#Features
1.Current Weather: Get the latest weather conditions for any location.
2.Hourly Forecast: View hourly weather forecasts for the next 48 hours.
3.Daily Forecast: Access a 7-day weather forecast to plan your week.
4.Search Locations: Search for weather information for any city or location.
5.Responsive Design: Enjoy a seamless experience on both desktop and mobile devices.
6.User-friendly Interface: Easy to navigate and understand.

#Usage
1.Open the iWeather web app in your web browser.
2.Enter the name of a city or location in the search bar.
3.Press the "Search" button or hit the "Enter" key.
4.View the current weather conditions, hourly forecast, and 7-day forecast for the selected location.

#Technologies Used
1.HTML: The structure of the web page.
2.CSS: Styling and layout.
3.JavaScript: Fetching weather data and user interactions.
4.OpenWeatherMap API: Used to retrieve weather data.
5.SVG Icons: Icons for weather conditions.
6.Responsive Design: Ensures a seamless experience on different devices.
